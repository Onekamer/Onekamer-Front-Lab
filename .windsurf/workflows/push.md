# Push Workflow

Pousse les derniers commits vers GitHub uniquement après confirmation manuelle.

## Étapes

1. Demande confirmation à l’utilisateur :
   > Souhaitez-vous pousser les derniers changements vers GitHub ? (oui/non)

2. Si la réponse est "oui", exécute :
   ```bash
   git push
   ```

3. Confirme le résultat :
   ✅ Push GitHub terminé avec succès.
